# Temporal-Graph-Neural-Network-For-Social-Bot-Detection
The project aims to track how users’ positions change in a social network over time using Graph Neural Networks and temporal analysis to identify key or suspicious activities.
## Dataset

This project uses the Twibot-20 dataset for social bot detection.
Download dataset from Kaggle:
https://www.kaggle.com/datasets/marvinvanbo/twibot-20

Note: Due to GitHub file size limitations, the dataset is not included in this repository.
